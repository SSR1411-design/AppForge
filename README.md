## 📌 Overview

AppForge takes a plain English description of an application and transforms it into a structured, executable schema including:

- UI schema
- API endpoints
- Database schema
- Authentication rules
- Validation report

It uses a multi-stage pipeline powered by LLMs to ensure structure, consistency, and reliability.
